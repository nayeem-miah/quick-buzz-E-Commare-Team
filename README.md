## **ğŸ“Œ Backend Setup**

```bash
git clone https://github.com/nayeem-miah/quick-buzz-E-Commare-Team.git
cd server
npm install
```

Create a `.env` file:

```
MONGO_URI=your_mongo_connection_string
JWT_SECRET=your_jwt_secret
PORT=3000

STORE_ID=your_ssl_store_id
STORE_PASS=your_ssl_store_pass

```

Start server:

